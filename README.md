# Thesis Defense Practice App

This simple Flask web application helps you practice defending your thesis. It shows random defense questions and provides a timer so you can rehearse your answers.

## Setup

1. Create a Python virtual environment (optional but recommended).
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the App

```bash
python app.py
```

Open your browser at [http://localhost:5000](http://localhost:5000) to start practicing.
